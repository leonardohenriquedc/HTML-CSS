Links Externo

Links externo e quando um link ao qual você colocou no seu site te leva para um outro site que não o mesmo do atual.

Para adicionarmos um link a uma frase ou para palavra (tipo aquelas que fica azul em varios sites), usamos a teg "a".
Ex:

Este canal do youtube e muito bom.

Temos um problema no site acima, pois quando clicamos no link o site ao qual está no link sobrepos sobre o nosso site logo a chance do usuario final nunca mais voltar ao seu site e muito grande para isso usamos dentro da teg "a" a função "target="_blank" que diz para o navegador criar uma pagina em branco para o link mantendo assim a sua pagina aberta.
Ex:

Você pode acessar o canal do youtube para mais informações.

Usando Links Internos

Links internos são aqueles que fazem parte do mesmo servidor, diretorio ou pertence a você mesmo. Nestes casos não e nescessario usar o "target="_blank"" para criar outra guia para o link afinal, o usuario tecnicamente não saiu do seu servidor.

Ex:
se quiser saber mais sobre lista click aqui

Temos a teg "rel" dentro da teg "a" ela serve para os mecanismos de buscar se localizarem com mais facilidade dentro do da estrutura do nosso site, usamos sempre que tem um link externo a nomeclatura external dentro da teg "rel" e tem outras como next para dizer ao mecanismo de busca que aquele link e "a proxima pagina" (geralmente usado para links interno), temos prev que indica que aquela pagina e a "anterior", e temos as nofollow usada para links que não tem o seu "aval" pode se dizer assim e tem o dofollow usada geralmente para links patrocinados.

UM AVISO IMPORTANTE: E sempre bom aprender linux pois a maioria dos servidores que hospedam site usam este tipo de sistema operacional.
Se por acaso vc quiser colocar um link interno que não esteja na pasta do site principal se for preciso voltar uma pasta use "../" se caso precisar avancar obviamente use "/"

Links para downloads

Para usar um link para download usamos a mesma teg "a" e adicionamos o arquivo que queremos porem, dentro da teg temos que adiciona a função "download="arquivo"" e o type="" para o navegador reconhe-los como dowlnloads.
Ex:

Se quiser baixar o meu tcc clique aqui

Se quiser em zip clique aqui
