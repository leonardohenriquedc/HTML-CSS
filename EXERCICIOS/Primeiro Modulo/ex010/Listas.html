Listas / organização

Listas Ordenadas

Temos atualmente no HTML 5 uma forma de organizar listas de forma automatica, mais especificamente falando a enumeração automatica dos itens.
Para isso temos a teg "ol" que dentro dela adicionamos outra teg nessa vez seguinda do item que deseja coloca na lista a teg e "li", onde para cada item se coloca dentro de um "li".
Segue um exemplo:

  1. Frango
  2. Picanha
  3. Lasanha

Podemos também definir a forma que será enumerado a lista, se você quiser ela em algarismos romanos ou em letras podemos adicionar a teg "type" dentro da teg "ol", logicamente se a pessoa deseja iniciar de outro valor que não por exemplo "1" no caso da enumeração numerica, pode-se adicionar junto de type uma teg "start" e colocar a parte de qual valor deseja prosseguir.
Segue abaixo um exemplo:

  1. Frango
  2. Lasanha
  3. Peru
  4. Batata

Lembrando que: a teg "start" deve conter apenas numeros.
Ex: Se for uma enumeração em letras e deseja iniciar a partir da Letra "E" você não pode colocar literalmente a letra e sim a numeração que ela se encontra, ou seja a letra "E" e a 5ª letra a partir do "A" logo você irar colocar o numero cinco.

Listas não Ordenadas

Podemos também fazer listas não ordenas em sequência, sendo ela a teg "ul". Segue exemplo:

  • Pão
  • Manteiga
  • Frango
  • Lasanha

Podemos também trocar o simbolo de sequencia também adicionando a teg "type" dentro de "ul".

  • Lasanha
  • Pão
  • Frango
  • Manteiga

Adicionando listas dentro de listas

As vezes precisamos adicionar listas para organizar varias coisas nas nossas tarefas, mas e quando precisamos fragmentar um item de uma lista e fazer dentro deste item outra lista? E oq vamos fazer
Para adiocionar uma lista dentro de outra e muito simples basta usar um "ol" ou "ul" adicionar um "li" (no caso o item) e dentro desse "li" adicionar outro "ol" ou "ul".
Ex:

  1. Liguagens de programação
    • Javascript
    • PHP
    • Python
  2. Liguagens de desenvolvimento de sites
    • HTML
    • CSS

Listas de definição

Listas de definição servem como se fossem um "dicionario". Se usa a teg "dl" para iniciar a lista (e fecha-la obviamente), dentro da teg temos o termo coma teg "dt" e temos temos a definição do termo com a teg "dd".
Ex:
